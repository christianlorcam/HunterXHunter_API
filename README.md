# Terminal de la Asociación de Cazadores - Hunter x Hunter

Este proyecto es una **interfaz de sistema de verificación de licencias** inspirada en el universo de *Hunter x Hunter*. Funciona como una terminal de consulta donde puedes buscar personajes icónicos de la serie mediante su ID oficial (Número de examen o código de amenaza).

## 🚀 Características
* **Sistema de Consulta:** Busca cazadores y criaturas por su ID único.
* **Base de Datos:** Integración de personajes principales, leyendas y amenazas del Continente Oscuro.
* **Modo Aleatorio:** Función de "Frecuencia Aleatoria" para descubrir registros de forma aleatoria.
* **Diseño Inmersivo:** Estética de terminal militar inspirada en la tecnología de la Asociación de Cazadores.
* **Soporte Offline:** Uso de imágenes locales para un rendimiento rápido.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura del sistema.
* **CSS3:** Estilos personalizados con diseño tipo "Terminal".
* **JavaScript (ES6+):** Lógica del sistema, manejo de eventos y simulación de API con Promesas.
* **WebP:** Formato de imagen optimizado para carga rápida.
